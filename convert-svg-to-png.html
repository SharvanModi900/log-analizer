SVG to PNG Converter

Instructions

This tool helps you convert your icon.svg file to PNG format, which you can then convert to ICO for the taskbar icon.

Conversion Steps

  1. Click the "Convert to PNG" button below
  2. Right-click on the converted image and select "Save Image As..."
  3. Save the file as "icon.png" in your project root directory
  4. Convert the PNG to ICO using an online tool
  5. Save the ICO as "build/icons/win/icon.ico"

Important Note

The new icon design without text will appear significantly larger and cleaner in the taskbar compared to the previous version with text.
